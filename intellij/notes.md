

# Common Changes That Made to Intellij

## Keymaps

- "Delete Line": ⌘D
- "Delete to Line Start": ⌘ Backspace
- "Duplicate Line or Selection": ⌘⇧C
- "Move Line Down": ⌥↓
- "Move Line Up": ⌥↑
- "Search Everywhere": ⌘O
- "Sort Lines": ⌘⇧S

diff in editor? ⌘⇧D
